Quero desenvolver um jogo mobile em Flutter (Dart) para Android, inspirado no estilo de jogos como Pou, com foco em cuidar de um gatinho virtual enquanto o usuário aprende inglês.

🎯 Objetivo do app

Criar um jogo fofo, interativo e educativo, onde o jogador cuida de um gato e interage com diferentes cenários, cada um com uma funcionalidade, incluindo aprendizado de inglês.

⚙️ Requisitos técnicos
Desenvolvido em Flutter (Dart)
Funcionar 100% offline
Sem backend próprio
Sem necessidade de banco de dados externo (Firebase opcional, mas evitar se possível)
Armazenamento apenas local (ex: shared_preferences ou similar)
Projeto organizado para versionamento no GitHub
🐱 Tela principal (Home)
Exibir:
O desenho do gatinho (fornecido por mim)
Um plano de fundo fofo
Barra inferior com ícones (estilo Pou)
Cada ícone leva para uma nova tela (usando navegação no Flutter)
📱 Navegação e Telas
🍔 Cozinha (ícone hambúrguer)
Cenário de cozinha
Interação de alimentar o gato
☁️ Quarto (ícone nuvem)
Cenário de quarto
Função de dormir
Incluir sistema de histórias em inglês:
Usuário deve avançar lendo/entendendo frases simples
Pode ter botões como “next”, “listen”, etc.
⭐ Área externa (ícone estrela)
Cenário ao ar livre
Interações leves (ex: brincar, animações)
📖 Sala de aula (ícone livro)
Sistema de aprendizado de inglês:
Palavras básicas
Quiz simples (múltipla escolha)
Associação de palavras com imagens
🌸 Passeio (ícone flor)
Novo cenário com a “gatinha”
Interação social simples
🐾 Mini game (ícone pata)
Um jogo dentro do app (mini game)
Estilo parecido com Pac-Man
Simples, com movimentação e coleta de itens
🎨 Design
Estilo fofo (cute/kawaii)
Cores suaves e amigáveis
Animações simples (ex: gato piscando, reagindo)
Interface intuitiva para iniciantes
🧩 Estrutura do código
Separação por telas (screens)
Componentes reutilizáveis (widgets)
Gerenciamento de estado simples (ex: setState ou Provider)
Código limpo e comentado
🎁 Objetivo final

O jogo será um presente, então quero que:

Seja bonito e bem organizado
Fácil de expandir no futuro
Divertido e educativo para aprender inglês
